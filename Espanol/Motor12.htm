Motor de Vida Artificial: Motor 12. Preparativos

Se tomó el Motor 11 y se le hicieron las siguientes mejoras:

  1. Se optimizaron varias instrucciones para mayor velocidad.
  2. Nuevas clases para hacer más fácil comprender el programa.
  3. Se adicionaron mas comentarios al código.
  4. Se verificó que pudiera compilarse tanto en ambiente Windows (usando Visual C++) como en Linux (usando g++).
  5. Se facilita el código para que las nuevas simulaciones (usando multi-entradas y multi-salidas) sean fácilmente implementadas.
  6. Se simplifican algunas clases.
